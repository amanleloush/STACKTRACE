## Problem
Given a string of `()[]{}`, decide if every opener is matched by a same-type closer in correct nesting order.

## Intuition
A stack matches the LIFO grammar of nested brackets perfectly. Push every opener as you see it; when you see a closer, the most recently opened bracket must match it ‚Äî peek the stack and pop. **Invariant**: at every point in the scan, the stack holds the prefix of openers that have not yet been closed, with the most recent on top. If the stack is ever empty when a closer arrives, or the top doesn't match, the string is invalid. If the stack is non-empty at the end, you have unmatched openers.

## Algorithm
Stack of openers with a closer‚Üíopener match table.

```python
def is_valid(s):
    match = {')': '(', ']': '[', '}': '{'}
    stack = []
    for ch in s:
        if ch in '([{':
            stack.append(ch)
        else:
            if not stack or stack[-1] != match[ch]:
                return False
            stack.pop()
    return not stack
```

## Walkthrough
Validate `"({[]})"`.

1. `(` ‚Äî push. Stack: `[(]`.
2. `{` ‚Äî push. Stack: `[(, {]`.
3. `[` ‚Äî push. Stack: `[(, {, []`.
4. `]` ‚Äî top `[` matches, pop. Stack: `[(, {]`.
5. `}` ‚Äî top `{` matches, pop. Stack: `[(]`. Then `)` ‚Äî top `(` matches, pop. Stack empty. Return `true`.
